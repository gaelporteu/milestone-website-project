A small one-pager project which introduced me to HTML and CSS. It is here where I learned how to structure and build using HTML. Rogue Pickings Responsive Website

HTML 5 & CSS

Here is where I began to create responsive content using CSS and media queries to resize the page depending on the viewport size. Also managed using version control through Git which also alows me to access the site and make changes from anywhere in the world as long as I have access to a computer and the web.
